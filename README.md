# Proyecto de Consulta COVID-19 en Colombia

## 1. Información General
Este software consume la API de Socrata para visualizar datos actualizados del Instituto Nacional de Salud.

## 2. Estructura del Proyecto (Arquitectura)
El proyecto está dividido en los siguientes componentes:
* **api/**: Contiene la lógica de conexión y consulta a la API de Datos Abiertos.
* **ui/**: Maneja la visualización de los datos en consola mediante tablas formateadas.
* **main.py**: Script principal que coordina la ejecución del programa.

## 3. Tecnologías Utilizadas
* Python
* 3 Librería Pandas (para manejo de datos)
* Librería Sodapy (para conexión con la API)
